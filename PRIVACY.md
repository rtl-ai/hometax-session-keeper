# Privacy Policy

Hometax Session Keeper does not collect, store, sell, transmit, or share user data.

The extension runs only on HomeTax domains:

- `https://hometax.go.kr/*`
- `https://www.hometax.go.kr/*`
- `https://*.hometax.go.kr/*`

It observes HomeTax session-timeout UI signals in the current page so it can click the official session extension prompt and show an approximate remaining-time badge when HomeTax exposes a timer value.

The extension does not:

- read cookies;
- read browser storage;
- read passwords, resident registration numbers, certificates, tax forms, uploaded files, or account data;
- send network requests to external servers;
- include analytics, tracking, advertising, or remote code.

HomeTax is operated by the Korean National Tax Service. This project is independent and is not affiliated with or endorsed by the National Tax Service.

Because automatic session extension keeps a login session active for longer, do not use this extension on public or shared computers. Log out of HomeTax when finished.
