# Support

For bugs, use GitHub Issues and include:

- browser and version;
- extension version;
- whether the issue happened on Chrome/Chromium, Firefox MV3, Firefox MV2, or Safari source build;
- a redacted description of the HomeTax page state;
- whether the browser action badge showed a time value.

Do not attach unredacted HomeTax screenshots, certificate dialogs, personal identifiers, cookies, or tax documents.
